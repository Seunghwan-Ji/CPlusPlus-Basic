# 비주얼 스튜디오 단축키 목록
| 단축키                          | 설명                                                         |
|---------------------------------|-------------------------------------------------------------|
| Ctrl + k + c 또는 Ctrl + /      | 지정한 구문 주석                                             |
| Ctrl + k + u 또는 Ctrl + /      | 지정한 구문 주석 해제                                        |
| Alt + Drag                      | 커서로 드래그한 범위만 수정                                  |
| ---                             |                                                             |
| F5                              | 디버깅 시작(중단점이 여러 개일 때 다음 중단점까지 코드 실행) |
| F9                              | 중단점 생성 및 해제                                         |
| F10                             | 디버깅 중, 구문 수행(한 줄 단위 코드 수행)                  |
| F11                             | 디버깅 중, 구문 수행(한 단계씩 코드 수행, ex) 함수)         |
| Shift + F5                      | 디버깅 중, 디버깅 종료하기                                  |
| ---                             |                                                             |
| F12                             | 선택한 코드의 선언, 정의된 부분으로 이동                    |
| Ctrl + Shift + Space            | 함수의 매개변수 정보 표시                                   |
| Ctrl + '=' + '.' (이후 엔터)    | 헤더 파일에 선언한 함수 정의 만들기                         |
| Ctrl + 'r', Ctrl + 'r'          | 프로젝트 내 이름이 같은 변수들의 이름 한 번에 바꾸기        |
| Ctrl + '\' + 'E'                | 오류 목록 보기                                              |
