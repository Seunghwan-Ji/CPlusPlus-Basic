# 단축키
- 지정한 구문 주석: Ctrl + k + c 또는 Ctrl + /
- 지정한 구문 주석 해제: Ctrl + k + u 또는 Ctrl + /
- 커서로 드래그한 범위만 수정: Alt + Drag

- 디버깅 시작: F5(중단점이 여러개일 때 다음 중단점까지 코드 실행)

- 중단점 생성 및 해제: F9
- 디버깅 중, 구문 수행(한 줄 단위 코드 수행): F10
- 디버깅 중, 구문 수행(한 단계씩 코드 수행, ex) 함수): F11
- 디버깅 중, 디버깅 종료하기: Shift + F5

- 선택한 코드의 선언, 정의된 부분으로 이동: F12

- 함수의 매개변수 정보 표시: Ctrl + Shift + Space

- 헤더 파일에 선언한 함수 정의 만들기: Ctrl + '=' + '.' (이후 엔터)

- 프로젝트 내 이름이 같은 변수들의 이름 한번에 바꾸기: Ctrl + 'r', Ctrl + 'r'

- 오류목록 보기: Ctrl + '\' + 'E'
