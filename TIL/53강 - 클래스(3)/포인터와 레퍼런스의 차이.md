# 포인터(pointer)
## 정의:
- 메모리 주소를 저장하는 변수입니다.
## 특징:
1. 초기화 필요:
- 포인터를 사용하려면 다른 변수의 주소로 초기화해야 합니다.
- 포인터는 nullptr 또는 유효한 메모리 주소로 초기화해야 안전합니다.

2. 연산 가능:
- 포인터는 주소를 저장하므로 주소 연산이 가능합니다(예: ++, -- 연산으로 주소 이동).

3. 명시적 역참조:
- * 연산자를 사용해 포인터가 가리키는 값을 접근합니다.

4. 동적 메모리 관리:
- 포인터는 동적 메모리 할당(malloc, new)과 해제(free, delete)에 주로 사용됩니다.

5. null 가능:
- 포인터는 nullptr(또는 NULL)을 가질 수 있습니다. 이는 포인터가 어떤 객체도 가리키지 않는다는 뜻입니다.
## 문법 예제:
`int x = 10;`   
`int* p = &x;` p는 x의 주소를 저장.   
`*p = 20;`     p가 가리키는 값을 변경.   
   
`std::cout << x;` 20   

# 레퍼런스(reference)
## 정의:
- 변수의 별칭(alias)입니다. 다른 변수와 동일한 메모리 공간을 공유합니다.
## 특징:
1. 초기화 필수:
- 레퍼런스는 선언할 때 반드시 초기화해야 합니다.
- 선언 이후 다른 변수를 참조할 수 없습니다(불변성).

2. 연산 불가능:
- 레퍼런스는 주소 연산이나 메모리 이동을 할 수 없습니다.

3. 역참조 불필요:
- 레퍼런스를 통해 값을 사용할 때 별도의 연산자가 필요 없습니다. 원래 변수처럼 사용합니다.

4. null 불가능:
- 레퍼런스는 반드시 유효한 변수를 참조해야 하며, nullptr을 가질 수 없습니다.

5. 사용 용도:
- 함수 인자로 사용 시, 원본 데이터를 수정하거나 복사 비용을 줄일 수 있습니다.
- 상속 관계에서 다형성을 구현할 때 자주 사용됩니다.
## 문법 예제:
`int x = 10;`   
`int& ref = x;` ref는 x의 별칭   
`ref = 20;`     ref를 통해 x의 값을 변경   
   
`std::cout << x;` 20   

# 차이점 비교
| **특징**                  | **포인터**                              | **레퍼런스**                          |
|---------------------------|-----------------------------------------|---------------------------------------|
| **초기화 여부**            | 초기화하지 않아도 선언 가능(`nullptr`) | 선언 시 반드시 초기화 필요             |
| **값 변경**                | 다른 주소를 가리킬 수 있음             | 초기화 후 다른 변수 참조 불가           |
| **연산 가능 여부**         | 주소 연산 가능                         | 주소 연산 불가능                       |
| **역참조 필요 여부**        | `*`로 값을 접근해야 함                 | 역참조 없이 값 접근 가능               |
| **null 허용 여부**         | `nullptr` 가능                        | null 불가능                           |
| **주요 사용 목적**         | 동적 메모리 관리, 주소 연산            | 함수 인자 전달, 변수 별칭              |
| **메모리 사용 방식**        | 포인터가 저장된 주소 참조             | 실제 변수와 동일한 메모리 공간 사용     |

# 사용 사례
포인터 사용 예시:
- 동적 메모리 할당:
`int* p = new int(10);`   
`std::cout << *p;` 10   
`delete p;` 메모리 해제   
   
레퍼런스 사용 예시:
- 함수 인자 전달:
`void increment(int& ref) {
    ref++;
}`   
   
`int x = 10;`   
`increment(x);` x가 직접 증가   
`std::cout << x;` 11   

# 요약
- 포인터는 유연하며 주소와 메모리를 조작할 수 있는 강력한 도구입니다.
- 레퍼런스는 사용법이 직관적이고 안전성이 높은 대안으로, 원본 데이터를 직접 조작하거나 복사를 피하고자 할 때 유용합니다.
