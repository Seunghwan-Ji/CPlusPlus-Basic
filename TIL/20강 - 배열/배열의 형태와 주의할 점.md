# 배열 형태
ex)   
`   
int iArray[5] = { };
`   
|0| |0| |0| |0| |0|   
각 4바이트 정수 자료형으로 값이 0으로 초기화, 5묶음의 연속된 메모리 영역.   
인덱스는 0 ~ 4.   
   
# 인덱스 초과 문제
ex)   
`   
int iValue = 1;
`   
`   
iArray[5] = 10;
`   
|0| |0| |0| |0| |0| |1|   
                    iValue   
초과한 5번째 인덱스는 변수 iValue의 메모리 공간이기에 의도치 않게 해당 변수의 값을 변경하게 되는 문제가 발생한다.
