1 byte = 8 bit (8개의 비트로 구성된 하나의 바이트)
1 bit: 0 or 1 (2진 분류, 경우의 수 = 2)
	크기 단위:
		1024 byte = 1 KB (1024 = 2의 10제곱)
		1024 KB = 1 MB
		1024 MB = 1 GB
		1024 GB = 1 TB

		1 TB는 몇 byte일까?
		1 TB = 1024 GB
			 = 1024 * 1024 MB
			 = 1024 * 1024 * 1024 KB
			 = 1024 * 1024 * 1024 * 1024 byte
			 = 2의 40제곱 byte

8 bit -> 2의 8제곱 = 256가지의 경우의 수 존재

0 0 0 0 0 0 0 0 -> 숫자 0을 표현
1 1 1 1 1 1 1 1 -> 숫자 255를 표현

<이진수 개념>
이진수 00000000 = 0
이진수 00000001 = 1
이진수 00000010 = 2 (1에 1을 얹으면 0이되고 왼쪽으로 넘어감)
이진수 00000011 = 3
이진수 00000100 = 4
이진수 11111111 = 255 (8비트에서 모든 자리가 1)

따라서 0부터 255까지의 숫자 표현 가능.
이진수도 십진수와 똑같이 오른쪽 끝인 일의 자리에서 자리 올림을 함.
